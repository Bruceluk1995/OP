## 按主题快速定位

先读权威文件，只有需要另一个角度时才加载配套文件。

| 主题 | 权威文件 | 配套文件 |
|------|---------|---------|
| 爽点 | `plot-emotion-system.md` | 高潮式→`plot-core-methods.md`；打脸/装逼→`style-combat-face.md`；题材公式→`genre-writing-formulas.md` |
| 情绪模块 | 项目 `对标/{书名}/剧情/情绪模块.md` | 无对标或设计新模块时读 `plot-emotion-system.md` |
| 节奏 | 项目 `对标/{书名}/剧情/节奏.md` | 无对标或设计新节奏时读 `outline-rhythm.md` |
| 高潮 | `plot-core-methods.md` | `outline-rhythm.md` + `outline-methods.md` |
| 金手指 | `plot-special-topics.md` | `outline-conflict.md` |
| 感情线 | `character-relations.md` | `outline-conflict.md` + `style-combat-face.md` + `plot-special-topics.md` |
| 反转 | `reversal-toolkit.md` | `plot-core-methods.md` |
| 人物 | `character-basics.md` | `character-design-methods.md` + `character-relations.md` |
| 女频写作 | `female-audience-writing.md` | `genre-readers.md` + `character-relations.md` |
| 去 AI 味 | `anti-ai-writing.md` | `banned-words.md` + `quality-checklist.md` |
