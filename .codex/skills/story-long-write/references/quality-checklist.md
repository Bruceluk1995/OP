# 长篇连载质量门槛

用于 `story-long-write` 的章后与阶段审查。目标是判断读者为什么继续读，以及连续章节是否真的推进；不是用密度指标替代观感。

## 单章盲读

- 本章开头继承了什么未解决问题？
- 主角本章想取得什么具体结果？
- 哪个选择改变了选择、关系、信息、风险、代价或预期？
- 章末留下的是自然的新问题，还是强行截断？
- 删除本章后，后续是否几乎不受影响？若是，本章需要合并或重写。

## 阶段门槛

### 连载承诺

- 当前卷有清楚的近期目标，整书有更远的欲望或谜团。
- 新设定服务当前选择，不用百科说明推迟故事。
- 升级、恋爱、经营或调查的每一步都会改变下一步可选项。

### 因果与累积

- 结果会成为后续原因，人物能记住并回应过去的伤害、恩惠和选择。
- 伏笔是后续选择的条件，不只是作者埋下的名词。
- 重复型单元至少改变对手、限制、代价、关系或目标，不能只换地点和怪物。

### 中段换挡

- 连续使用同一解决机制前，必须出现适应、反制、代价或目标变化。
- 训练、会议、赶路、查账和说明只保留会改变局势的部分。
- 高潮不是把此前已经成立的结论再公开证明一次。

### 人物生命力

- 核心角色有不同欲望、恐惧、盲点、能力和羞耻点。
- 角色声音来自关注对象和判断逻辑，不靠口头禅。
- 主角的成功会产生新责任、敌人、关系成本或自我冲突。

### 叙述与行文

- 第一人称遵守知识边界并保持可识别的选择性叙述。
- 对话与描写承担行动，不替作者重复已经发生的事。
- 章节张弛按功能变化，不要求固定字数出现冲突或爽点。

## 返回工位

| 问题 | 处理 |
|---|---|
| 连载承诺或卷目标失效 | 回卷纲 / 故事核 |
| 连续章节无累积、重复换皮 | 重排阶段结构 |
| 关键章只有流程与说明 | 重写场景 |
| 事实矛盾 | 一致性修复 |
| 故事成立但文字僵硬 | 行文修改 / 去 AI 味 |

字数只用于排期和载体适配，不作为章节完成证明。
