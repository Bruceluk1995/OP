# Terminology And Localization

Avoid Chinese fantasy defaults in Japanese female-audience fantasy romance.

## Replace These Surfaces

- `修仙`, `修炼`, `境界`, `金丹`, `元婴` -> `魔法`, `魔力`, `加護`, `祝福`, `聖属性`, `魔導`.
- `灵气`, `灵根` -> `魔力`, `聖力`, `精霊の加護`.
- `宗门`, `门派`, `师尊` -> `魔法学院`, `神殿`, `騎士団`, `公爵家`, `侯爵家`.
- `丹药`, `符箓`, `法宝` -> `ポーション`, `護符`, `魔道具`, `聖具`.
- Chinese palace-wife hierarchy -> use `王妃`, `側妃`, `後妻`, `異母妹`, `宮廷派閥` only if the setting calls for it.

## Cultural Localization, Not Literal Translation

When adapting a Chinese 古言/宫斗/朝堂 script into Japanese female fantasy romance, do not preserve Chinese court-drama institutions unless the user explicitly wants a Chinese setting. Rebuild the social engine for the target lane.

Replace or reframe:

- `皇帝`, `皇上`, `圣上` -> `国王`, `王`, `皇帝陛下` only in an empire setting; otherwise use `王太子`, `王宮`, `王命`.
- `皇后`, `贵妃`, `妃嫔`, `后妃`, `后宫` -> `王妃`, `側妃`, `婚約者`, `愛妾`, `宮廷派閥`; avoid harem-ranking drama unless the setting truly needs it.
- `太子`, `皇子`, `公主`, `夺嫡` -> `王太子`, `第二王子`, `王女`, `王位継承争い`, `貴族派閥争い`.
- `嫡女`, `庶女`, `嫡庶之争` -> `公爵家の長女`, `正妻の娘`, `後妻の娘`, `異母妹`, `相続順位`, `家門の後継問題`.
- `宫女`, `太监`, `嬷嬷`, `内侍` -> `侍女`, `女官`, `侍従`, `執事`; avoid eunuch-based palace logistics unless the story is intentionally Chinese court fantasy.
- `冷宫`, `打入冷宫` -> `離宮に幽閉`, `修道院送り`, `塔への幽閉`, `王都追放`.
- `赐婚`, `圣旨`, `抄家`, `灭族` -> `王命による婚約`, `勅命`, `爵位剥奪`, `領地没収`, `一族追放`, `処刑判決`.
- `宫斗`, `后宫争宠`, `朝堂权谋` -> `社交界の失脚`, `宮廷派閥争い`, `貴族院の審問`, `神殿の告発`, `婚約破棄ざまぁ`, depending on the target lane.

Do not translate only the word. Translate the function: humiliation, inheritance pressure, marriage order, palace faction, public trial, exile, or revenge must become a Japanese female-fantasy social mechanism the viewer immediately understands.

## Prefer These Terms

`令嬢`, `婚約破棄`, `悪役令嬢`, `聖女`, `神殿`, `公爵`, `辺境伯`, `侯爵`, `王太子`, `騎士団長`, `魔導師`, `契約結婚`, `仮初めの婚約`, `王妃教育`, `社交界`, `領地`, `商会`, `加護`, `祝福`, `魔道具`.

## Prose Check

If a term would make the reader imagine Chinese cultivation or Chinese palace drama, replace it. If a mechanic would make the scene read like male-audience RPG leveling, translate it into social proof, magic etiquette, church authority, or noble-house consequence.
