# Topic Pack: Behavioral Risk And Scams

Use for gambling, sports betting, financial scams, insurance misunderstandings, consumer loans, sunk cost, loss aversion, FOMO, lotteries, probability, and misleading ads.

## Core Engine

- Start with the temptation: "almost won", "limited-time offer", "guaranteed yield", "everyone else is making money."
- Explain the math and the psychology together.
- Show how the game is designed before discussing individual mistakes.

## Required Payoffs

- One lure.
- One probability or incentive mechanism.
- One behavioral bias.
- One exit rule.

## Title Shapes

- `为什么赌球不是你懂球，而是庄家懂你？`
- `なぜ「あと少しで勝てた」は一番危ない錯覚なのか`
- `The House Does Not Need You To Be Stupid`
- `骗局最狠的地方，是让你觉得自己快赢了`

## Avoid

- Teaching exploit tactics.
- Mocking victims.
- Overclaiming criminal details without sources.
