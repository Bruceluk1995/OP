# 日本男频异世界题材包

## 读者等待

能力的可操作性、战术选择、身份错位、资源/掉落结果、怪物与世界规则、阶级变化、实用吐槽和更大舞台。先明确本篇主承诺，其他爽点只服务它。

## 能力与压力合同

- 能力边界不等于能力代价。默认先设计能力能带来的收益、扩张和爽点，再决定故事压力来自哪里。
- 不得为了显得“有深度”给每个外挂自动附加反噬、寿命/记忆损失、未来欠债、使用后失控或道德惩罚。只有用户选择诅咒、等价交换、黑暗能力或高风险路线时，能力本身的代价才是主发动机。
- OP、生产、经营与慢生活候选默认使用收益型能力。压力优先来自敌人策略、需求暴涨、资源竞争、保护对象、身份暴露、规模扩大、时间窗口、政治反应或成功后的新机会。
- 候选批次要有能力经济差异；不得把整批都写成“能力很强，但每次使用必须受罚”。限制可以是使用范围、信息未知或操作条件，也可以完全没有硬限制。

## 可用发动机

- 战斗升级：观察规则 -> 受阻/判断失误 -> 改战术 -> 可见结果 -> 新威胁。
- OP 支配：外界误判 -> 主角克制/布局 -> 压倒性结果 -> 阵营、领地或情报后果。
- 追放逆转：旧队误判一个真实功能 -> 一次决定性事件改变双方处境 -> 主角选择是否回头。
- 地城/领地：资源、地形与部下取舍改变战局或政治，不写经营报表。
- 慢生活：舒适感落在食物、工具、住处、关系和安全的具体变化，不强求公开打脸。

## 人物合同

主角要有实用欲望、可执行能力、一个会影响选择的偏见或习惯，以及日本男性向轻小说可读的自我判断。人物选择可以产生后果，但不得把这种后果机械地塞进能力规则。敌人会根据新信息改变策略；同伴不是欢呼器。

每当本路径创建具名角色、角色阵容或剧本时，读取 `../../jp-isekai/references/character-prompt-contract.md` 并同步交付角色提示词。女主与女性同伴默认采用可区分的日式异世界后宫动画美少女设计，不因职业、贫困、丧偶、育儿或创伤自动生成中年脸、疲惫体态或纪录片式工装。角色母提示词只锁定脸、发型、体型、服装和稳定佩饰；双手必须为空，不写象征物、武器、工具、技能特效、动作或场景。

## 本地化

使用自然的 `スキル`、`ステータス`、`ランク`、`冒険者ギルド`、`ダンジョン`、`魔物`、`装備`、`商会`、`領地` 等日式 RPG 机制。先写功能，再决定名词。避免宗门、修炼境界、朝堂、赘婿式家族制度的直译。

## 禁止默认

- 把爽文自动写成开店、合同、审计和群众排队；
- 为证明外挂连续做三次实验；
- 为每个外挂自动附加反噬、寿命/记忆损失、债务或失控惩罚；
- 敌人永久愚蠢，只负责震惊；
- 用ステータス面板替代战术与选择；
- OP 主角为了制造危机随机变弱；
- 战斗升级写成草药、料理和仓库流水账。

## 完结检查

能力/策略承诺已兑现；中段出现新机制；高潮由主角选择触发；资源/身份/关系/世界位置至少一项真正改变；结尾不是下一话预告。
