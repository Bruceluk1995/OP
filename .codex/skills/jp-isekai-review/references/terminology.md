# Banned Chinese Leakage

Flag these in Japanese-style isekai drafts unless the user requested Chinese fantasy:

`方玄|白石坊|灵|靈|妖|修炼|修煉|宗门|宗門|世家|筑基|金丹|元婴|玄|仙|龟|龜|灵根|灵气|妖兽|妖獸|灵兽|靈獸|坊市|洞府|丹药|丹藥|法宝|法寶|灵石|靈石|飞剑|飛劍|道侣|道侶`

Suggested replacements:

- Cultivation energy -> `魔力`, `魔素`, `マナ`
- Sect -> `ギルド`, `学院`, `騎士団`, `商会`, `教会`
- Beast contract -> `使い魔契約`, `従魔契約`, `テイム`
- Cultivation stage -> `ランク`, `レベル`, `階位`
- Spiritual item -> `魔道具`, `アーティファクト`, `魔石`, `ポーション`
