# 短篇正文退稿清单

这份清单用于 `story-short-write` 的正文盲读。它不是打分表：故事核失败、因果断裂或主角失去行动权时，格式、字数和文句不能抵消问题。

## 先做盲读

暂时不看大纲、对标、指标和作者说明，只读正文并回答：

- 第一屏具体承诺了什么，读者在等什么结果？
- 最早想退出的位置在哪里，为什么？
- 主角此刻想要什么，做了什么选择，选择改变了什么？
- 中段是否换了麻烦、关系或机制，还是围着同一结论重复证明？
- 结尾偿还了开头欠下的主要情绪债吗？

答不出来，先退回故事核或场景，不做句子润色。

## 五项不可抵消的门槛

### 1. 故事核成立

- 主角欲望、主要阻力、异常手段与代价能形成一条因果链。
- 去掉世界观名词后，冲突仍然具体。
- 结局主要由主角的选择造成，不靠突然降临的权威、巧合或反派自毁。

### 2. 每场戏真的改变状态

- 场景进入与退出时，至少一项发生不可忽略的变化：选择、关系、信息、风险、代价或读者预期。
- 已经证明过的结论不换道具再证明一次。
- 训练、实验、审查、赶路和解释只有改变局势时才保留。

### 3. 人物在做选择

- 主角不是被通知、被救、被安排着走完整篇。
- 配角有自己的利益和反应，不只负责惊讶、解释或递证据。
- 反派的行动能升级局势，不是等待主角公开打脸。

### 4. 第一人称或叙述声线成立

- 第一人称只说“我”能看到、听到、推断或事后明确得知的内容。
- 叙述者会挑选细节、下判断、误解或隐瞒；不是把第三人称稿换成“我”。
- 角色声音来自注意力和价值判断，不靠口头禅区分。

### 5. 读者债务得到偿还

- 开头制造的核心疑问、羞辱、欲望或危险获得结果。
- 证明一次即转向后果、反扑或新问题。
- 收束展示新状态，不复述主题，不用解释替代余味。

## 返回工位

| 最早失败点 | 返回 |
|---|---|
| 承诺模糊、结局靠运气 | 重做故事核 |
| 中段重复、因果次序错误 | 大改结构 |
| 场景只有手续和说明 | 重写场景 |
| 故事成立但语言僵硬 | 行文修改 / 去 AI 味 |
| 五项均成立 | 可交读者 |

长度是预算，不是完成证明。默认完整短篇约 12,000 字符；只统计，不以补足字数为理由增加无状态变化的内容。
