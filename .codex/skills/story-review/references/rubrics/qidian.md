# 起点男频审稿门槛

这是平台与题材适配补充，不能替代正文盲读，也不能用固定字数或章数自动判定作品好坏。

| 门槛 | 成立 | 阻断风险 |
|---|---|---|
| 连载期待 | 主线目标、成长可能或谜团让读者知道继续追什么 | 只有设定展示，没有近期行动目标 |
| 成长反馈 | 能力、资源、地位、认知或关系变化会改变下一步选择 | 升级只是数字变化，局势与打法不变 |
| 金手指机制 | 有限制、试错、后果与新的解法 | 每次自动解决问题，或只被旁白反复提及 |
| 冲突升级 | 对手会适应，代价与目标会换挡 | 同一打脸、刷怪、签到或围观循环换皮 |
| 章节推进 | 每章改变选择、关系、信息、风险、代价或预期 | 删除整章几乎不影响后续 |
| 男频回报 | 实力、尊严、秩序、掌控、探索或建设的回报可见 | 只靠围观惊叹宣布主角很强 |

审稿时引用具体章节与场景，最后返回 `故事核 / 卷纲结构 / 场景 / 一致性 / 行文` 中最早失败的工位。
