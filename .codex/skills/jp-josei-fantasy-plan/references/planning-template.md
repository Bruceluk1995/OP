# Planning Template

Use this template for a full concept package.

## Concept Package

- Japanese title candidates: 5-10 options with tag signals in the title.
- Project scale: short-season novel by default for episode-like/YouTube serials; fixed 6 episodes. Use chapter/episode prose files, not screenplay format.
- Selected lane: one subtype from the type-pack routing table, not generic "female fantasy".
- Benchmark priority used: project拆文 / live website benchmark card / project-local type pack / bundled type pack fallback.
- Loaded benchmark card: source, primary benchmark, contrast benchmarks, and differentiation rule.
- Loaded type pack: lane file and any project-local notes.
- Tag cluster: 3-5 primary tags and 1-2 secondary flavors.
- Reader promise: one sentence naming the emotional reward.
- Opening wound: what the heroine loses in chapter 1.
- Heroine: status, skill, false reputation, real strength, private fear, active goal.
- Male lead: title, public image, private wound, reason he notices her, concrete way he helps.
- Antagonists: fiance, sister, family, saint, court faction, church, or noble rival; include motive and evidence trail.
- World rules: noble hierarchy, church/magic rules, marriage law, inheritance, reputation economy.
- Romance ladder: 5-7 beats from fair recognition to public choosing.
- Zamaa ladder: 5-7 beats from accusation to evidence to consequence.
- Lane engine: the selected subtype's non-negotiable payoff loop.
  - Engagement annulment: accusation -> evidence -> witness/legal venue -> public consequence -> better match.
  - Villainess/otome: flag -> changed action -> route ripple -> recognition -> new route.
  - Saintess/church: false evaluation -> ritual/healing proof -> inquiry -> institutional consequence.
  - Contract marriage: clause -> forced proximity -> practical proof -> rule change -> chosen partner.
  - Frontier/duke: old rejection -> estate problem -> heroine competence -> public protection -> home.
  - Craft/job: product/service -> customer proof -> reputation or money movement -> respected romance.
  - Palace/family: record -> domestic/palace pressure -> legal or family witness -> chosen family.
  - Loop: known flag -> changed choice -> unexpected ripple -> hidden cause -> broken future.
  - Interspecies: culture rule -> misunderstanding -> nonhuman tenderness -> public/ritual choice.
  - Strong heroine: action proof -> status pressure -> respected authority -> public honor.
  - Letter comedy: message object -> wrong assumption -> social complication -> sincere re-choice.
- First season / episode list: fixed 6 episodes/chapters. Each item needs hook, heroine choice, romance movement, evidence/zamaa movement, payoff, cost/status change, and next hook.
- Happy-ending shape: marriage, restored title, new estate, chosen family, territory success, cleared name.
- Terminology map: source terms to Japanese female-fantasy terms.
- Banned leakage list: Chinese fantasy terms, male-audience RPG mechanics, or modern slang to avoid.

## Title Bias

Japanese female webnovel titles can be long and premise-forward. Prefer titles that reveal the setup, injustice, and reward:

- `婚約破棄された...`
- `悪役令嬢ですが...`
- `追放された聖女は...`
- `契約結婚のはずが...`
- `辺境伯に拾われた...`

Avoid title promises that the outline cannot pay off in the first 1-3 chapters.
