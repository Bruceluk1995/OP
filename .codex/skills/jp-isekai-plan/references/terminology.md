# Japanese Isekai Terminology Map

Use this when adapting Chinese fantasy into Japanese RPG isekai.

## Preferred Terms

- 灵气 / 霊気 -> 魔力, 魔素, マナ
- 修炼 -> レベル上げ, 訓練, 魔力を鍛える, スキルを育てる
- 妖兽 -> 魔物, 従魔, 使い魔, モンスター
- 灵兽 -> 使い魔, 従魔, 聖獣 only if rare/high fantasy
- 灵根 -> 適性, 魔力適性, スキル適性, 加護
- 宗门 -> 冒険者ギルド, 魔法学院, 騎士団, 教会, 商会
- 世家 -> 貴族家, 名家, 大商会
- 坊市 -> 鉱山町, 辺境町, 市場町, ギルド街
- 筑基/金丹/etc. -> Eランク/Dランク/Cランク, レベル, 階位
- 丹药 -> ポーション, 薬, 霊薬 if intentionally classical
- 法宝 -> 魔道具, アーティファクト, 装備
- 灵石 -> 魔石, 銀貨, 金貨, ギルドポイント

## Banned Unless User Explicitly Wants Chinese Style

方玄, 灵根, 灵气, 修炼, 妖兽, 灵兽, 宗门, 世家, 筑基, 金丹, 元婴, 仙, 玄, 洞府, 坊市, 丹药, 法宝, 灵石, 飞剑, 道侣.

## Localization Principle

Preserve plot function, not surface term. If a Chinese term means "elite resource monopoly", render it as guild/noble/company control. If it means "beast contract cultivation", render it as familiar/taming/mana-feedback mechanics.
