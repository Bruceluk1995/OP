# Market Snapshot: Japanese Male-Audience Isekai

Use this as a lightweight pattern note, not as fixed market truth. Refresh by browsing when the user asks for current rankings.

Observed from public ranking/tag pages around 2026-07-01:

- 小説家になろう rankings still surface 異世界転生/転移, チート, 魔法, ざまぁ, コメディ, ほのぼの, 勇者召喚, 巻き込まれ, ペットが最強, 料理/旅 patterns.
- Kakuyomu 異世界ファンタジー daily/男性向け pages show strong use of male protagonist, 固有スキル, ステータス, 生産系, ダンジョン, 迷宮, 成り上がり, 主人公最強, 追放/ざまぁ, ハーレム-adjacent, 飯テロ, and スローライフを目指したい.
- A common male-audience engine is: unfair low status -> odd skill -> experiment -> public proof -> practical reward -> new trouble.
- A common slow-life engine is: protagonist wants peace or a base -> cheat creates economic value -> guild/nobles/neighbors notice -> trouble interrupts the peaceful plan.
- Popular surface vocabulary is Japanese RPG fantasy, not cultivation: ランク, スキル, ステータス, 魔力, 魔法, 冒険者, ギルド, 魔物, ダンジョン, 貴族.
